## ◈ Cos'è Viral Decoder?

**Viral Decoder** è un'applicazione React avanzata progettata per creatori di contenuti che vogliono smettere di indovinare e iniziare a vincere. Alimentata dall'incredibile velocità di **Groq** e l'intelligenza di **Qwen 2.5**, l'app analizza video virali e genera un piano d'azione (blueprint) per replicarne il successo.

Non è un semplice analista: è un **Direttore Creativo** che ti dice esattamente cosa filmare, cosa dire e perché funziona.

## ⚡ Caratteristiche Principali

- **🔥 Viral Core**: Individuazione immediata dell'elemento chiave che ha reso il video virale.
- **🎬 Production Blueprint**: Uno storyboard scena-per-scena (0-3s, 3-10s, ecc.) per la tua produzione.
- **📈 Creative Strategy**: Consigli strategici personalizzati per la tua nicchia.
- **🎣 Hook Analysis**: Analisi profonda dei primi secondi per massimizzare la ritenzione.
- **📊 Radar Chart**: Visualizzazione grafica di 5 metriche chiave (Hook, Retention, Emotion, Trend, Shareability).

## 🛠️ Stack Tecnologico

- **Frontend**: React 18 + Vite
- **AI Engine**: Qwen 2.5 su Groq (Free Tier)
- **Inference**: Groq LPU (Low Latency)
- **Design**: Sistema CSS Premium con Glassmorphism e Cyber-Aesthetics.

## 🚀 Setup Rapido

### 1. Clona e Installa
```bash
git clone https://github.com/CosmoNetinfo/Viral-Decoder.git
cd viral-decoder
npm install
```

### 2. Configura Groq (Completamente Gratis)
1. Ottieni una chiave API gratuita su [Groq Cloud](https://console.groq.com/).
2. Crea un file `.env` nella cartella radice:
   ```env
   VITE_GROQ_API_KEY=La_Tua_Chiave_API_Qui
   ```

### 3. Avvia
```bash
npm run dev
```

## 📖 Utilizzo

1. Seleziona la **piattaforma** (TikTok, YouTube, Reels, ecc.).
2. Incolla l'**URL** di un video virale.
3. Clicca su **ANALIZZA**.
4. Ricevi il tuo manuale operativo per creare il prossimo video virale.
