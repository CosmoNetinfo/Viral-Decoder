## ◈ Cos'è Viral Decoder?

**Viral Decoder** è un'applicazione React avanzata progettata per creatori di contenuti che vogliono smettere di indovinare e iniziare a vincere. Alimentata da **Google Gemini 1.5 Flash**, l'app analizza video virali in tempo reale e genera un piano d'azione (blueprint) per replicarne il successo.

Grazie al **Google Search Grounding**, l'app non solo analizza il contenuto, ma recupera statistiche reali per aiutarti a studiare i numeri dietro la viralità.

## ⚡ Caratteristiche Principali

- **📊 Real Stats Report**: Recupero di views, likes, commenti e engagement rate reali tramite Google Search.
- **🔥 Viral Core**: Individuazione immediata dell'elemento chiave che ha reso il video virale.
- **🎬 Production Blueprint**: Uno storyboard scena-per-scena (0-3s, 3-10s, ecc.) per la tua produzione.
- **📈 Creative Strategy**: Consigli strategici personalizzati per la tua nicchia.
- **📊 Radar Chart**: Visualizzazione grafica di 5 metriche chiave.

## 🛠️ Stack Tecnologico

- **Frontend**: React 18 + Vite
- **AI Engine**: Google Gemini 1.5 Flash (Free Tier)
- **Data Source**: Google Search Retrieval (Grounding)
- **Design**: Sistema CSS Premium con Glassmorphism e Cyber-Aesthetics.

## 🚀 Setup Rapido

### 1. Clona e Installa
```bash
git clone https://github.com/CosmoNetinfo/Viral-Decoder.git
cd viral-decoder
npm install
```

### 2. Configura Gemini (Completamente Gratis)
1. Ottieni una chiave API gratuita su [Google AI Studio](https://aistudio.google.com/).
2. Crea un file `.env` nella cartella radice:
   ```env
   VITE_GEMINI_API_KEY=La_Tua_Chiave_API_Qui
   ```

### 3. Avvia
```bash
npm run dev
```

## 📖 Utilizzo

1. Seleziona la **piattaforma** (TikTok, YouTube, Reels, ecc.).
2. Incolla l'**URL** di un video virale.
3. Clicca su **ANALIZZA**.
4. Ricevi il tuo manuale operativo per creare il prossimo video virale.
